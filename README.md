# 🏧 ATM Management System

A Python-based ATM Management System developed using Object-Oriented Programming (OOP) concepts. This project simulates basic banking operations such as checking account balance, depositing money, withdrawing money, changing PINs, viewing transaction history, and managing multiple accounts.

## 🚀 Features

* Secure Account Login
* Check Account Balance
* Deposit Money
* Withdraw Money
* Change ATM PIN
* Transaction History Tracking
* Multiple Account Support
* Menu-Driven Interface
* Object-Oriented Design

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* Dictionaries
* Lists
* Loops & Conditional Statements

## 📸 Functionalities

### 1. User Authentication

* Login using Account Number and PIN.
* Validates user credentials before granting access.

### 2. Balance Inquiry

* Displays current account balance.

### 3. Deposit Money

* Allows users to deposit money into their account.
* Updates account balance instantly.

### 4. Withdraw Money

* Allows users to withdraw funds.
* Prevents withdrawal when balance is insufficient.

### 5. Change PIN

* Verifies current PIN.
* Allows users to update their PIN securely.

### 6. Transaction History

* Records deposits and withdrawals.
* Displays complete transaction history.

### 7. Multiple Accounts

* Supports multiple bank accounts using Python dictionaries.

## ▶️ How to Run

1. Install Python 3.
2. Clone the repository:

git clone <repository-url>

3. Navigate to the project folder:

cd ATM-Management-System

4. Run the application:

python atm.py
